## Usage
1. In the `root` directory:
    ```git
    $ git clone https://github.com/GallenHu/hexo-theme-Daily.git themes/Daily
    $ npm install --save hexo-renderer-sass
    ```

2. Change the `theme` property in the `config.yml` file.
    ```yml
    # theme: landscape
    theme: Daily
    ```

3. Run: `hexo server`
